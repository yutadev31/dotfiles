# Dotfiles

## Installation

```sh
./install.sh cli # Install CLI tools only
./install.sh desktop # Install hyprland desktop (includes CLI tools)
```

## Features

### System

- **Distro**: [Arch Linux](https://archlinux.org/)
- **Kernel**: [Linux](https://www.kernel.org/)
- **Boot Loader**: [GRUB](https://www.gnu.org/software/grub/)
- **Shell**: [Fish](https://github.com/fish-shell/fish-shell)

### Desktop Environment

- **WM**: [Hyprland](https://github.com/hyprwm/Hyprland)
- **Bar**: [Waybar](https://github.com/Alexays/Waybar)
- **Launcher**: [Rofi](https://github.com/davatorium/rofi)
- **Notification**: [Swaync](https://github.com/ErikReider/SwayNotificationCenter)
- **Input Method**: [Fcitx5](https://github.com/fcitx/fcitx5)

### Applications

- **Terminal**: [Alacritty](https://github.com/alacritty/alacritty)
- **Editor**: [Neovim](https://github.com/neovim/neovim)
- **System Monitor**: [Btop](https://github.com/aristocratos/btop)
- **Browser**: [Google Chrome](https://www.google.com/intl/ja_jp/chrome/)
- **File Manager**: [Nemo](https://github.com/linuxmint/nemo)
- **Media Player**: [VLC](https://github.com/videolan/vlc)
